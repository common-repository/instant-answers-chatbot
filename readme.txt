Embed an AI-powered chatbot created with Instant Answers into your WordPress site seamlessly.

== Description ==

Instant Answers is an innovative AI-powered platform that enables users to quickly craft chatbots using their existing content. By uploading a document or providing a website URL, users can design chatbots capable of addressing questions pertinent to that content.

With this plugin, you can effortlessly integrate the chatbot crafted on Instant Answers directly into your WordPress website, providing a state-of-the-art support experience for your visitors.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/instant-answers-chatbot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the "Instant Answers Options" screen to configure the plugin by adding your chatbot ID.

== Frequently Asked Questions ==

= How do I set up my chatbot? =

Once you've designed your chatbot on Instant Answers, simply copy its chatbot ID. Navigate to the settings page on your WordPress dashboard and paste the ID there to integrate the chatbot into your website.

= What if my chatbot isn't responding? =

Ensure the chatbot ID is correctly copied and pasted into the settings page. If problems persist, please visit the Instant Answers support page or contact us directly.
